# 📊 Análise de Vendas com Power BI

## 🔍 Sobre o projeto

Este projeto tem como objetivo analisar o desempenho de vendas utilizando dados estruturados em múltiplas tabelas, simulando um ambiente real de negócio.

A análise foi desenvolvida no Power BI a partir de uma base em Excel, com foco em gerar insights sobre produtos, vendedores e volume de vendas ao longo do tempo.

---

## 🗄️ Estrutura dos Dados

A base foi organizada em formato relacional, contendo as seguintes tabelas:

* **Clientes**: informações dos clientes
* **Vendedores**: dados dos vendedores
* **Produtos**: catálogo de produtos
* **Vendas**: registro das vendas realizadas
* **ItensVendas**: detalhamento dos itens por venda

---

## 🔗 Modelagem de Dados

Os dados foram modelados no Power BI utilizando relacionamentos entre as tabelas, permitindo análises integradas.

A tabela **ItensVendas** atua como fato principal, conectando-se às demais dimensões:

* Produtos
* Vendas
* Clientes
* Vendedores

---

## 📈 Análises desenvolvidas

* Valor total de vendas por produto
* Quantidade vendida por período
* Performance de vendedores
* Comparação mensal de vendas

---

## 🛠️ Ferramentas utilizadas

* Power BI
* Excel
